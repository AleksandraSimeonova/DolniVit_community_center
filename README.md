# DolniVit_community_center

## Description
The website of the Dolni Vit Community Center aims to promote the center's activities, events, and cultural initiatives, while providing public access to important documents, announcements, and information for the local community.


## Tech Stack
   - [x] Vanila JS
   - [] Firebase for backend, hosting
   - [x] Tailwind CSS with UI components
   - [x] Page.js for routing
   - [x] Lit-html for templating
   - [x] Local Dev server with Vite
   - [x] Vite Bundling with vite
